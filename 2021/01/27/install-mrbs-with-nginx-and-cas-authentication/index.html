安装MRBS

前半部分主要参考的是MRBS的官方安装文档。

  1. 从 https://sourceforge.net/projects/mrbs/files/ 下载MRBS，同时，需要保证机器上已经安装了nginx, mysql/mariadb, fcgiwrap, php-fpm, php-xml 和 php-curl。

  2. 解压MRBS的压缩包。在本文中，MRBS被安装在了/var/www/mrbs/中。
